# CafeSales-Analysis

## Overview
This project transforms a messy, real-world cafe transactions dataset into actionable business strategy. The raw data arrived with missing values, placeholder errors, and broken formatting. I built an end-to-end pipeline to clean the data, analyze it with SQL, and build a high-impact dashboard

---

## Technical Stack
* **Python (Pandas):** Data cleaning, validation, and error correction
* **SQL (SQLite & DBeaver):** Business logic and trend analysis
* **Power BI:** Interactive executive dashboards
* **Excel:** Quick pivot tables and data verification
